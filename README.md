# ERP Telegram + IA (PostgreSQL) — Web Service (Render)

## Suba no GitHub estes arquivos/pastas:
- server.js
- package.json
- render.yaml (opcional)
- src/ (pasta inteira)
- *.example.json (opcional)

## Variáveis obrigatórias no Render
- DATABASE_URL
- TELEGRAM_BOT_TOKEN
- TELEGRAM_ADMIN_ID
- OPENAI_API_KEY
- CRON_TOKEN
- PUBLIC_BASE_URL (se usar webhook)

## Start
Render:
- Build: `npm install`
- Start: `npm start`
